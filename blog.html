1) What is DOM and what is the purpose of DOM?

Ans: The full meaning of DOM is D-Document, O-Object, M-Model, in short (DOM). It represents that how a web browser work with e web page internally. Besides this, DOM determines what content should be on e page and how each element of the content relates to the other elements.
The main purpose of using DOM is that it allows us to create, change or remove elements from a website documents.It's normally manupulate our data and return values


2) How will you select HTML elements using DOM? Name the DOM methods.

Ans: To select HTML elements we can use DOM selectors. there are mainly five effective selectors in DOM which are:
1) getElementById() - the selection is based on the Id name
2) getElementsByClassName - This selectors returns all the elements that matches the specified Class name
3) getElementsByTagName - It returns all the elements that matches the specified Tag name
4) querySelector - This one returns the first element that matches the specified CSS selector in the document.
5) querySelectorAll - It returns all the elements that match the specified CSS selector in the document.
Beside this some another methods are: getAttribute(), setAttribute(), appendChild(), removeChild(), createElement() etc.


3) What is event bubble?

Ans: When an event happens on an element, it first runs the handlers on it, then on its parent, then all the way up on other elements. In example: if we have three specific events like child, parent and grandparent. so first child will work , then parent and then grandparent and that's how it works.


4) What is a callback function and why will you use it?

Ans: Basically a callback function is a function which we can use again or later. when we use a function as an argument of another function, then we can called it callback function.
When we want one function to execute only after another function has completed its execution, we use callback functions in JavaScript. It needs to pass as a parameter to other functions to make a function callback.
